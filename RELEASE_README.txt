============================================
Color Blindness Accessibility Checker
Version 0.1.0
============================================

A desktop widget to check if your color palettes and images are accessible 
to people with color blindness.

QUICK START
-----------
1. Double-click "install.bat"
2. Follow the prompts
3. Launch from desktop or Start Menu
4. Done!

FEATURES
--------
- Test multiple colors at once (HEX, RGB, RGBL formats)
- Upload images or take screenshots
- 8 different color blindness simulations
- Modern, rounded interface
- Fast and lightweight

MANUAL INSTALLATION
-------------------
If you prefer not to use the installer:
1. Right-click "colorblind-widget.exe"
2. Select "Create shortcut"
3. Move the shortcut to your desktop
4. (Optional) Right-click shortcut → "Pin to taskbar"

UNINSTALLATION
--------------
Run "uninstall.bat" to remove all shortcuts.

SYSTEM REQUIREMENTS
-------------------
- Windows 10 or later (64-bit)
- No additional software required
- ~15 MB disk space
- Minimal RAM usage

HOW TO USE
----------
Color Palette Tab:
  1. Enter colors in HEX (#FF0000), RGB (255,0,0), or RGBL format
  2. Click "Add" to add colors to your palette
  3. Select different color blindness types from dropdown
  4. See how all colors appear to different users

Image Analysis Tab:
  1. Click "Load Image" or "Take Screenshot"
  2. Select a color blindness simulation
  3. See how your image looks in real-time

COLOR BLINDNESS TYPES
----------------------
- Protanopia (Red-blind) - ~1% of males
- Deuteranopia (Green-blind) - ~1% of males
- Tritanopia (Blue-blind) - Very rare
- Protanomaly (Red-weak) - ~1% of males
- Deuteranomaly (Green-weak) - ~5% of males
- Tritanomaly (Blue-weak) - Very rare
- Achromatopsia (Complete color blindness) - Very rare

DESIGN TIPS
-----------
✓ Don't rely solely on color - use labels, patterns, icons
✓ Ensure high contrast (WCAG AA: 4.5:1 ratio)
✓ Test with multiple simulations
✓ Avoid red/green, blue/purple combinations
✓ Upload screenshots of your actual designs

TROUBLESHOOTING
---------------
Q: Application won't start
A: Ensure you have Windows 10 or later. Try running as administrator.

Q: Screenshot feature doesn't work
A: Grant screen capture permissions in Windows Privacy Settings.

Q: Icon doesn't show
A: Right-click shortcut → Properties → Change Icon → Select from .exe

Q: How do I uninstall?
A: Run uninstall.bat or manually delete the shortcuts.
